# Digital Ateljè - Art by Sandra
En modern, minimalistisk konstnärsportfolio/hemsida för att visa konstverk digitalt. 
Byggd med Next.js och fokus på en “gallery experience” med möjlighet att kontakta konstnären. 

# Tech Stack;
- Next.js
- React
- Tailwind CSS

# Om projektet;
Detta projekt är ett examensarbete där jag skapat en hemsida som ska fungera som en "digital ateljé" för mitt egna konstnärliga skapande. Sidan är just nu endast en frontend gjord i NextJS och Tailwind CSS. Hemsidan fungerar som en portfolio där man kan se olika konstverk, få en inblick i mitt skapande och ta kontakt med konstnären vid intresse. Målet var att skapa en snygg hemsida som var relativt simpel men ändå höll en kreativ tråd. Färg, komposition och andra detaljer är medvetet valda och sidan är responsiv och därmed fullt fungerande i mobilvy. Utöver detta innehåller sidan andra detaljer och funktioner såsom en modal som simulerar möjligheten att signa upp sig för nyhetsbrev, en cookiebanner där man godkänner cookies, formulär som simulerar att man kan kontakta konstnären via mail, en digital vernissage/förhandsvisning av verken som finns i galleriet med mera. Vid vidareutveckling av detta projekt är det tänkt att koppla på en backend som ersätter viss funktion som nu är gjort i fronten och även addera yttligare funktionalitet.
